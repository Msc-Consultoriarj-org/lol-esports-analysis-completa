# Dados e Recursos

## Arquivos de Dados

| Formato | Descricao | Localizacao |
|---------|-----------|-------------|
| **Parquet** | Dados binarios otimizados | `Data/parquet/` |
| **JSON** | Estatisticas agregadas | `Data/JSON/` |
| **CSV** | Dados brutos (nao incluidos no git) | `Data/CSV/` |

## Estrutura de Dados

```
Data/
├── CSV/                  # Dados brutos por ano (excluidos do git)
├── JSON/                 # Estatisticas agregadas por ano
│   └── lol_esports_YEAR.json
├── Markdown/             # Relatorios anuais
│   └── lol_esports_YEAR.md
├── Ligas/                # Analises por liga
│   ├── LCK_2026.md
│   ├── LEC_2026.md
│   ├── LPL.md
│   └── ...
├── Meta/                 # Dados de meta de campeoes
│   └── Champions.md
├── parquet/              # Formato binario otimizado
│   └── lol_esports_YEAR.parquet
└── INDEX.md              # Este arquivo
```

## Scripts Python

| Script | Funcao |
|--------|--------|
| `consolidate_data.py` | Consolida todos os CSVs |
| `create_all_files.py` | Gera Parquet, JSON e Markdown |
| `exploratory_analysis.py` | Analise exploratoria |
| `detailed_analysis.py` | Analise detalhada da estrutura |
