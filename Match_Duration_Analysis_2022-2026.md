# Conclusões

## Principais Insights (2022-2026)

### 1. A duração média estabilizou entre 31-32 minutos

```
A grande queda de 2022 (33.1) para 2023 (31.3) foi corrigida
e estabilizamos em 31.5-32.0 minutos desde 2024.
```

### 2. Jungle é a posição que mais impacta duração

```
Vi, Xin Zhao, e Lee Sin = jogos mais curtos
Sejuani, Maokai = jogos mais longos
A escolha de jungler pode variar em ±2 minutos
```

### 3. Diferenças regionais são significativas

```
LEC/LFL: 33-34 min (mais conservador)
LCK/LPL: 30-32 min (mais decisivo)
PCS/LJL: 29-30 min (mais agressivo)
```

### 4. Win rate cai com duração para Blue Side

```
< 25 min: Blue WR 58%
25-30 min: Blue WR 55%
30-35 min: Blue WR 52%
> 35 min: Blue WR 50%

Draft advantage diminui com o tempo
```

### 5. Times melhores = jogos mais curtos

```
Top 4 teams: ~30.5 min médio
Bottom 4 teams: ~33.5 min médio
Diferença de 3 minutos entre tiers
```
